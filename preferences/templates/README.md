# Templates

Reusable formats for common document types. The context router loads these automatically when the trigger matches.

| Template | Trigger | Purpose |
|----------|---------|---------|
| [app-readme.md](app-readme.md) | Creating/editing app README | Frontmatter + protocol structure for app directories |
| [automation-spec.md](automation-spec.md) | Designing a new automation | Success criteria, validation, architecture, error handling |
| [journal_template.md](journal_template.md) | Drafting journal entry | Structured session log format (frontmatter + sections) |
| [project_template.md](project_template.md) | Creating project | Project README with frontmatter (status, tags, next-steps) |
