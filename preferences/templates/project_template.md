# Project Template

## File Structure

```
projects/{domain}/YYYY-MM-DD_brief-description/
├── README.md          # Goal, status, progress, next steps
├── docs/              # Optional: research, specs, decisions
├── scripts/           # Optional: automation code
├── configs/           # Optional: configuration
└── backups/           # Optional: point-in-time snapshots
```

## README Format

```markdown
---
project: brief-name
status: in-progress
created: YYYY-MM-DD
tags: [domain, topic]
---

# Project: Title

## Goal

What this project achieves and why.

## Status

- [ ] Step 1
- [ ] Step 2
- [ ] Step 3

## Current Progress

What's been done so far.

## Next Steps

What's next.
```
