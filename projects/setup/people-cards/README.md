# Setup: Build Your People Directory

## Goal

Create persistent context about the important people in your life so your AI doesn't need to ask twice.

## Steps

- [ ] **Your profile** — Create `knowledge/people/{your-name}/README.md` with bio, goals, and key context
- [ ] **Partner/spouse** — If applicable, create their card with relationship context
- [ ] **Children/dependents** — If applicable, create cards with developmental/health context
- [ ] **Close contacts** — Create 3-5 cards for people you mention frequently
- [ ] **Professional contacts** — Create cards for key colleagues, clients, or collaborators

## Card Format

Each person gets a directory: `knowledge/people/{firstname-lastname}/README.md`

```markdown
**Relationship:** friend | colleague | family | partner
**Added:** YYYY-MM-DD

## Notes
- [preference] Prefers email over phone
- [work] VP Engineering at Company
- [family] Two kids, ages 8 and 11
- [gift] Loves craft coffee
- [connection] Introduced through mutual friend
```

## Tips

- Start small — create cards as people come up naturally
- The AI will suggest creating cards when you mention new people
- Include whatever helps the AI be more contextually aware
