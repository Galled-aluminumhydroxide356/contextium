# Host Docs Map

Reference mapping of documentation hosting across your infrastructure.

## Purpose

Tracks which documentation and knowledge bases are hosted where, so you know where to look for specific information.

## Setup

1. Create a mapping table below with your documentation sources
2. Update as you add or move documentation

## Template

| System | Host | URL | Content |
|--------|------|-----|---------|
| Hudu | Cloud/Self-hosted | `https://your-hudu-instance` | IT documentation, client info |
| Wiki | Self-hosted | `https://your-wiki` | Internal knowledge base |
| This repo | Local | `knowledge/` | Personal knowledge, health, finance |
| Confluence | Cloud | `https://your-confluence` | Team documentation |

## Use Cases

- Quick reference for finding where specific docs live
- Onboarding new team members to documentation landscape
- Identifying documentation gaps or redundancies
