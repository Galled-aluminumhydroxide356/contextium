# Monarch

Personal finance tracking for accounts, transactions, and budgets.

## Requirements

- Monarch Money account
- Authentication credentials

## Setup

1. Create or log in to your Monarch Money account
2. Store your credentials in your vault
3. Link your financial accounts within Monarch
4. Configure API access

## Key Data

| Data | Use |
|------|-----|
| Account balances | Net worth tracking |
| Transactions | Spending categorization |
| Budgets | Budget vs. actual |
| Cash flow | Income and expense trends |

## Use Cases

- Checking account balances and net worth
- Reviewing recent transactions
- Tracking budget adherence
- Generating personal finance summaries
