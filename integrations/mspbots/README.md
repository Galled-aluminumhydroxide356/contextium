# MSPBots

MSP-specific analytics platform for business metrics and KPI dashboards.

## Requirements

- MSPBots account
- API credentials

## Setup

1. Obtain API credentials from your MSPBots account
2. Store credentials in your vault
3. Configure data source connections within MSPBots

## Key Data

| Resource | Use |
|----------|-----|
| Dashboards | Pre-built KPI views |
| Reports | Custom report generation |
| Metrics | Business performance data |
| Alerts | Threshold-based notifications |

## Use Cases

- Pulling business metrics for weekly reviews
- Generating KPI reports
- Monitoring service desk performance
- Tracking revenue and profitability metrics
