# Strety

EOS (Entrepreneurial Operating System) platform for scorecards, rocks, and to-dos.

## Requirements

- Strety account with API access
- API credentials

## Setup

1. Obtain API credentials from your Strety account
2. Store credentials in your vault
3. Identify your team and scorecard IDs

## Key Data

| Resource | Use |
|----------|-----|
| Scorecards | Weekly KPI metrics |
| Rocks | Quarterly goal tracking |
| To-Dos | Action items from meetings |
| Issues | IDS (Identify, Discuss, Solve) list |

## Use Cases

- Pulling weekly scorecard data for review
- Tracking rock completion progress
- Preparing for weekly L10 meetings
- Reviewing to-do completion rates
