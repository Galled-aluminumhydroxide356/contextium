# Gemini

AI research agent for web research, content summarization, and external tasks.

## Requirements

- Gemini CLI installed
- API key configured

## Setup

1. Install the Gemini CLI
2. Configure your API key
3. See `agent-configs/claude/GEMINI.md` for delegation rules

## When to Delegate to Gemini

- Web research requiring browsing and summarization
- Content analysis from URLs
- Tasks requiring real-time web access
- Todoist operations (task creation, updates)

## Key Capabilities

| Task | Example |
|------|---------|
| Web research | "Research the latest on topic X and summarize" |
| Content summary | "Summarize this article: URL" |
| Task management | "Create a Todoist task for..." |
| Fact checking | "Verify this claim with current sources" |

## Use Cases

- Gathering research for project decisions
- Summarizing long articles or documents
- Managing tasks through natural language
- Getting current information beyond training data
